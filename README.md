🍕 Pizza Sales Analysis Dashboard - Power BI
------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
📌 Project Overview

This project showcases an interactive Pizza Sales Dashboard built in Power BI to analyze sales performance and customer behavior.

The focus of this dashboard is not only on visualizing data but also on identifying patterns like top-selling products, peak order timings, and sales distribution across categories and sizes.
_________________________________________________________________________________________________________________________________________________________________________________________________
🎯 Objectives

• Evaluate overall sales and revenue performance

• Identify high and low performing pizzas

• Analyze ordering patterns based on time

• Understand customer preference by size and category

• Support business decisions using data insights

📂 Dataset Features

The dataset includes:

• Order ID

• Pizza Name

• Category

• Size

• Quantity

• Price

• Order Date & Time

📊 Dashboard Insights

This dashboard provides meaningful insights such as:

💰 Total Revenue analysis with trend understanding

🍕 Identification of best-selling and least-selling pizzas

📊 Category-wise contribution to total sales

⏰ Peak ordering hours and customer activity patterns

📦 Size-wise performance to understand demand

🛠️ Tools & Technologies Used

• Power BI

• Power Query (for data cleaning and transformation)

• DAX (for calculated measures and insights)

🚀 Features

• Interactive slicers for dynamic filtering

• Clean and structured dashboard layout

• Drill-down capability for deeper analysis

• Easy-to-understand visuals for quick insights
